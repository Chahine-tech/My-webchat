## Installation

Install dependencies with npm or yarn


https://user-images.githubusercontent.com/70902783/145604727-49a9b3cb-1f55-4a53-9a11-73e251cc0853.mp4



```bash
npm install
```
To test the server part :
```bash
npm start
```

Compile the code (for the frontend):
```bash
npm run build
```
